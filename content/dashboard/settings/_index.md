# Settings

## Globale Einstellungen für das Erscheinungsbild des Dashboards
Hier können globale Einstellungen für das Erscheinungsbild aller Dashboards festgelegt werden.

{{< image src="settings.png" alt="Settings" title="Settings" >}}

Dies umfasst die folgenden Parameter:

### ![1](../1.png) Backgroundcolor Grid
Legt die Hintergrundfarbe des Dashboards fest (![1](../1.png)). Mit dem zweiten Schieberegler können auch RGBA-Farben erstellt werden, d. h. Hintergründe mit halbtransparenten Effekten. Um den Hintergrund vollständig transparent zu machen, kann die Option "**Set Backgroundcolor transparent**" (![2](../2.png)) aktiviert werden.

{{< image src="color-picker.png" alt="Farbauswahl" title="Farbauswahl" >}}

### ![2](../2.png) Set Backgroundcolor transparent
Um den Hintergrund vollständig transparent zu machen, kann die Option "**Set Backgroundcolor transparent**" aktiviert werden.

### ![3](../3.png) Backgroundcolor Widget
Legt die Hintergrundfarbe des Widgets fest.

### ![4](../4.png) Bordercolor Widget
Legt die Rahmenfarbe des Widgets fest.

### ![5](../5.png) Borderwidth Widget
Legt die Rahmenstärke des Widgets fest.

### ![6](../6.png) Borderradius Widget
Legt fest, wie stark die Ecken der Widgets abgerundet dargestellt werden sollen.

### ![7](../7.png) Margin Widget
Legt den Abstand zwischen den einzelnen Widgets fest.

## Energy dashboard app Version
Die aktuelle Version der App wird unter "**App-Version**" (![8](../8.png)) angezeigt.
