# Energyflows

## Energyflow hinzufügen

1. Um einen Energiefluss zu erstellen, gehen Sie zum Energyflow-Bereich und klicken Sie unten links auf die Schaltfläche "**Add Energyflow**" (![1](../1.png)).

{{< image src="add-energyflow.png" alt="Energyflow hinzufügen" title="Energyflow hinzufügen" >}}

2. Geben Sie im Dialogfeld einen Namen für den Energiefluss ein, den Sie erstellen möchten (![1](../1.png)), und klicken Sie auf die Schaltfläche "**Add Energyflow**" (![2](../2.png)).

{{< image src="add-new-energyflow.png" alt="Energyflow hinzufügen" title="Energyflow hinzufügen" >}}

## Energyflow verwalten

{{< image src="energyflow-buttons.png" alt="Energyflow Buttons" title="Energyflow Buttons" >}}

### ![1](../1.png) bearbeiten
Klicken Sie hier, um den Energyflow zu bearbeiten. Weitere Informationen finden Sie im Abschnitt **Energyflow bearbeiten**.

### ![2](../2.png) duplizieren
Klicken Sie hier, um den Energyflow zu duplizieren. Dem Namen des duplizierten Energyflows wird "**(Kopie)**" hinzugefügt.

### ![3](../3.png) exportieren
Klicken Sie hier, um den Energyflow zu exportieren. Ihr Browser wird versuchen, eine Datei mit dem Inhalt des Energyflows herunterzuladen.

### ![4](../4.png) ansehen
Klicken Sie hier, um den Energyflow in Ihrem Browser anzuzeigen.

### ![5](../5.png) importieren
Klicken Sie hier, um einen zuvor exportierten Energyflow erneut zu importieren. Dem Namen des importierten Energyflows wird "**(Import)**" hinzugefügt.

## Energyflow bearbeiten

### Energyflow
Durch Klicken auf den Pfeil neben "**Energyflow**" können die Parameter für den Kreis in der Mitte angezeigt werden.

#### Height
Gibt die Höhe des gesamten Energieflusselements an.

#### Width
Gibt die Breite des gesamten Energieflusselements an.

#### Minimum Height
Gibt die Mindesthöhe des gesamten Energieflusselements an.

#### Minimum Width
Gibt die Mindestbreite des gesamten Energieflusselements an.

#### Title
Gibt einen Titel an, der frei innerhalb des Energieflusses platziert werden kann.

#### Font Size Title
Legt die Schriftgröße des Titels fest.

#### Bold Title
Wenn diese Option ausgewählt ist, wird der Titel fett dargestellt.

#### Color Title
Legt die Schriftfarbe des Titels fest.

#### Offset Title X
Gibt den Versatz auf der X-Achse der Titelposition relativ zur Mitte des gesamten Energieflusses an.

#### Offset Title Y
Gibt den Versatz auf der Y-Achse der Titelposition relativ zur Mitte des gesamten Energieflusses an.

#### Radius
Gibt den Radius aller Kreise an.

#### Dot size
Gibt den Radius der animierten Punkte an.

#### Length
Gibt die Länge aller zwischen den Kreisen angezeigten Linien an.

#### Stroke
Legt die Rahmendicke der Kreise und die Dicke aller Linien fest.

#### Position
Gibt die Position des ersten Kreises an. Folgende Positionen sind verfügbar:  
`0`: links - oben  
`1`: Mitte - oben  
`2`: rechts - oben  
`3`: links - Mitte  
`4`: Mitte – Mitte  
`5`: Mitte - rechts  
`6`: links - unten  
`7`: Mitte – unten  
`8`: rechts - unten  

#### Offset Postion X
Dieser Wert wird zur X-Koordinate der ausgewählten Position addiert.

#### Offset Postion Y
Dieser Wert wird zur Y-Koordinate der ausgewählten Position addiert.

#### Icon
Zeigt das Symbol des ersten Kreises an.

#### Offset Icon X
Dieser Wert wird zur X-Koordinate der Symbolposition addiert.

#### Offset Icon Y
Dieser Wert wird zur Y-Koordinate der Symbolposition addiert.

#### Scale Factor Icon
Das Symbol wird um diesen Faktor vergrößert oder verkleinert.

#### Icon Label
Gibt eine frei positionierbare Beschriftung für die aktuelle Last an.

#### Font Size Label
Legt die Schriftgröße der Beschriftung fest.

#### Bold Label
Wenn diese Option ausgewählt ist, wird die Beschriftung fett dargestellt.

#### Color Label
Legt die Schriftfarbe der Beschriftung fest.

#### Offset Label X
Dieser Wert wird zur X-Koordinate der Beschriftungsposition addiert.

#### Offset Label Y
Dieser Wert wird zur Y-Koordinate der Beschriftungsposition addiert.

#### Icon Value
Der Wert der hier ausgewählten Gruppenadresse wird als Text im Energiefluss angezeigt.

#### Decimal Places Value
Anzahl der Stellen nach dem Komma.

#### Font Size Value
Legt die Schriftgröße des Wertetextes fest.

#### Bold Value
Wenn diese Option ausgewählt ist, wird der Wert fett dargestellt.

#### Color Value
Legt die Schriftfarbe des Wertetextes fest.

#### Icon Value Unit
Die Werteeinheit besteht aus einem frei definierbaren Text, der hinter dem Symbolwert angezeigt wird.

#### Offset Value X
Dieser Wert wird zur X-Koordinate der Position des Symbolwerts addiert.

#### Offset Value Y
Dieser Wert wird zur Y-Koordinate der Position des Symbolwerts addiert.

#### Color Icon
Legt die Farbe des Symbols fest.

#### Color Circle
Legt die Farbe des Kreises um das Symbol fest.

#### Backgroundcolor Circle
Legt die Hintergrundfarbe des Kreises fest.

#### Border Circle Style
Legt den Stil der Kreislinie fest, ob sie als durchgezogene Linie, oder als gestrichelte/gepunktete Linie angezeigt werden soll.

#### Scale Factor Radius
Dieser Wert wird mit dem Radius multipliziert, sodass jeder Kreis einzeln im Verhältnis zum Standardradius vergrößert oder verkleinert werden kann.

#### Backgroundcolor
Legt die Hintergrundfarbe des gesamten Energieflusselements fest.

#### Set Backgroundcolor transparent
Durch Auswahl dieser Option wird der Hintergrund des Energieflusses vollständig transparent.

### Load 01 - 16
Durch Klicken auf eine der Button mit den Nummern 1 bis 16 kann der aktuellen Last eine untergeordnete Last hinzugefügt werden. Last Nr. 1 befindet sich vertikal nach oben, danach sind die Lasten im Uhrzeigersinn um die Last angeordnet.  

Jede neue Last kann wiederum bis zu 16 untergeordnete Lasten haben, und für jede Last stehen die folgenden Parameter zur Verfügung.  

Die Parameter können entweder durch Klicken auf den Pfeil oder durch direktes Klicken auf den Kreis in der Vorschau auf der linken Seite angezeigt werden.

Durch Klicken auf das Mülleimer-Symbol wird die ausgewählte Last und alle untergeordneten Lasten gelöscht.

#### Address
Das hier ausgewählte Objekt wird verwendet, um die Animation des Punktes zu berechnen, der sich entlang der einzelnen Linien bewegt.  
`Value < 0`: Der Punkt bewegt sich in eine Richtung  
`Value = 0`: Der Punkt ist ausgeblendet  
`Value > 0`: Der Punkt bewegt sich in die entgegengesetzte Richtung  

#### Zero Offset
Definiert einen Bereich um den Wert 0 (`Wert` < `0 – Zero Offset` und `Wert > 0 + Zero Offset`), in dem der Punkt ausgeblendet wird.

#### Icon
Legt das Symbol des Kreises fest.

#### Offset Icon X
Dieser Wert wird zur X-Koordinate der Symbolposition addiert.

#### Offset Icon Y
Dieser Wert wird zur Y-Koordinate der Symbolposition addiert.

#### Scale Factor Icon
Das Symbol wird um diesen Faktor vergrößert oder verkleinert.

#### Icon Label
Gibt eine frei positionierbare Beschriftung für die aktuelle Last an.

#### Bold Label
Wenn diese Option ausgewählt ist, wird die Beschriftung fett dargestellt.

#### Font Size Label
Legt die Schriftgröße der Beschriftung fest.

#### Color Label
Legt die Schriftfarbe der Beschriftung fest.

#### Offset Label X
Dieser Wert wird zur X-Koordinate der Beschriftungsposition addiert.

#### Offset Label Y
Dieser Wert wird zur Y-Koordinate der Beschriftungsposition addiert.

#### Icon Value
Der Wert der hier ausgewählten Gruppenadresse wird als Text im Energiefluss angezeigt.

#### Decimal Places Value
Anzahl der Stellen nach dem Komma.

#### Bold Value
Wenn diese Option ausgewählt ist, wird der Wert fett dargestellt.

#### Font Size Value
Legt die Schriftgröße des Wertetextes fest.

#### Color Value
Legt die Schriftfarbe des Wertetextes fest.

#### Icon Value Unit
Die Werteeinheit besteht aus einem frei definierbaren Text, der hinter dem Symbolwert angezeigt wird.

#### Offset Value X
Dieser Wert wird zur X-Koordinate der Position des Symbolwerts addiert.

#### Offset Value Y
Dieser Wert wird zur Y-Koordinate der Position des Symbolwerts addiert.

#### Color Icon
Legt die Farbe des Symbols fest.

#### Color Line
Legt die Farbe der Linie fest.

#### Color Dot
Legt die Farbe des Punktes fest.

#### Color Circle
Legt die Farbe des Kreises um das Symbol fest.

#### Backgroundcolor Circle
Legt die Hintergrundfarbe des Kreises fest.

#### Invert
Kehrt die Logik der Punktanimation um und ändert somit die Richtung für positive/negative Werte.

#### Remove Circle
Blendet den Kreis um das Symbol aus.

#### Border Circle Style
Legt den Stil der Kreislinie fest, ob sie als durchgezogene Linie, oder als gestrichelte/gepunktete Linie angezeigt werden soll.

#### Line Style
Legt den Stil der Linie fest, ob sie als durchgezogene Linie, oder als gestrichelte/gepunktete Linie angezeigt werden soll.

#### Duration
Gibt die Zeit an, die ein Punkt benötigt, um sich von einem Ende der Linie zum anderen zu bewegen.

#### Scale Factor Line
Dieser Wert wird mit dem Parameter Length multipliziert, sodass jede Linie einzeln relativ zur Standardlänge vergrößert oder verkleinert werden kann.

#### Scale Factor Radius
Dieser Wert wird mit dem Parameter Radius multipliziert, sodass jeder Kreis einzeln im Verhältnis zum Standardradius vergrößert oder verkleinert werden kann.

Sobald alle Parameter konfiguriert sind, kann der Energiefluss durch Klicken auf die Schaltfläche "Save **Energyflow**" gespeichert werden.
