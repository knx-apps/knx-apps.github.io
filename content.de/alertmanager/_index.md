# Alertmanager App

## Installieren der App
Die App kann über eine **\*.ipk** Datei installiert werden. 

### Installation über IPK Datei
In diesem Fall haben Sie eine Datei mit der Erweiterung **\*.ipk** aus einer vertrauenswürdigen Quelle erhalten. Greifen Sie über die Startseite Ihres Controllers auf den 
Marktplatz zu:

{{< image src="install1.png" alt="Dashboard" title="Dashboard" >}}

Klicken Sie auf das Hamburger-Menü und wählen Sie **Installieren von Datei**.

{{< image src="install2.png" alt="Dashboard" title="Dashboard" >}}

Wählen Sie im Popup-Menü die **\*.ipk** Datei aus und klicken Sie auf **OK**, um die Installation durchzuführen.

{{< image src="install3.png" alt="Dashboard" title="Dashboard" >}}
